# Spotify Weather + News Playlists 🎵🌦️📰

This project recommends Spotify playlists based on **real-time weather** and **current news sentiment**.  
The idea: music that adapts to the world around you.  

## Features
- Fetches live weather data (OpenWeather API)
- Fetches top news headlines (NewsAPI)
- Performs basic sentiment analysis on headlines
- Maps weather + news → Spotify playlist link

## Example
☀️ Sunny + Positive News → Upbeat Pop Playlist  
🌧️ Rainy + Negative News → Chill Acoustic Playlist
